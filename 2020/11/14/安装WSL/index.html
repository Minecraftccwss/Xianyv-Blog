WSL安装指南

想要知道怎么安装WSL，首先要先知道WSL是什么

微软官方给出的说明是 “基于Windouws的Linux子系统”

说白了也就是 能在Windouws上运行的Linux系统

话说Windouws的终端也够丑的，没有颜色，新版的PowerShell虽然比较符合Linux用户的使用习惯，但是缺点太多了，比如：

  1. 启动慢(机械硬盘用户非常有感触)

  2. 报错太长了,生怕别人看不懂

  3. 部分程序还会提示 因为在此系统上禁止运行脚本,所以无法执行

  4. …

所以你现在知道为什么有这么多人不喜欢Widouws的终端了吧

讲了这么多,也应该教你们怎么安装了


打开设置,搜索 “启用或关闭 Windows 功能”

找到”使用于Linux的Windouws子系统”

安装完成之后重启电脑，打开 Microsoft Store，搜索你要的分发版，安装

然后在开始菜单里找到它并运行，就会有半分钟（SATA固态测试）左右的安装（注册）时间

完了
